# App-icon design archive

`Keyroost icon design.zip` is the source design archive for the keyroost app
icon — both colorways plus the editable source. It is reference / design
material only: it is not part of the Learn site's pages and nothing on the site
links to it. The keyroost application icons used in builds are generated from
this source separately.

Kept here as a convenient design archive. Safe to ignore when reading the site.
